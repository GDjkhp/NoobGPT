# Bot Commands

## Node.js Computer #1
### Prefix: -

* `-ask [prompt]` OpenAI GPT-3.5-Turbo (ChatGPT) chat completion.
* `-imagine [prompt]` OpenAI Dall-E image generation.
* `-quote` Make it a quote. (Source: replied message)

## Python Computer #2
### Prefix: -

* `-bard [prompt]` Google Bard chat completion.
* `-anime [query]` Search and stream anime using Gogoanime.
* `-search [query]` Search and stream TV shows and movies using Actvid.
* `-tic` Play tic-tac-toe with someone.

## Node.js Computer #3
### Prefix: !!

* `!!p [query/link]` Play something. Supports YouTube and Spotify.
* `!!np` Now playing.
* `!!q` Show query.
* `!!s` Skip track.
* `!!help` Show music commands help. [(More info)](https://discord-musicbot.gdjkhp.repl.co/)

## Python Computer #4
### Slash commands

* `/new (optional: [rated = True/False] [opponent = user])` Create a new chess game invite.
* `/cancel` Cancel the created invite.
* `/accept` Accept the invite.
* `/decline` Decline the invite.
* `/cmove [move]` Move pieces. (UCI recommended)
* `/surrender` Surrender and lose the current match.
* `/show` Show current game.
* `/stats (optional: [user])` Show the stats of the player.
* `/leaderboard` Show top chess players.
